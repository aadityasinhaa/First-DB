# Your First DB App

A minimal browser-based IndexedDB application for managing customer records.

## Features

- **Load DB** — Initialize the IndexedDB database with sample customer data
- **Query DB** — Retrieve and display all stored customer records
- **Clear DB** — Remove all records from the database
- Real-time operation log with timestamps
- Clean, responsive UI with neutral color palette

## Tech Stack

- Vanilla HTML, CSS, and JavaScript
- IndexedDB API for client-side storage

## Getting Started

1. Clone the repository
2. Open `index.html` in a modern browser
3. Use the control buttons to interact with the database

No build tools or dependencies required.

## Browser Support

Requires a browser with IndexedDB support (all modern browsers).
